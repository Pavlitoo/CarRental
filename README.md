# 🚗 CarRental — Сучасна інтелектуальна система оренди авто

Професійний веб-додаток на базі **Django**, розроблений як курсовий проєкт. Система автоматизує повний цикл прокату автомобілів: від вибору моделі до автоматичної перевірки документів за допомогою комп'ютерного зору.

## 🌟 Ключові особливості

* **🤖 AI Age Verification (KYC):** Автоматичне розпізнавання дати народження з фото паспорта за допомогою **Tesseract OCR**. Система самостійно блокує користувачів молодше 18 років.
* **💎 Loyalty System:** Розумний кешбек (Bronze 5%, Silver 7%, Gold 10%) та накопичувальний баланс.
* **🛠 Automated Maintenance:** Система відстежує кількість поїздок кожного авто та автоматично знімає машину з публікації після 5 замовлень для проходження ТО.
* **🎟 Promo Codes:** Гнучка система купонів зі знижками та лімітами використань.
* **📊 Admin Dashboard:** Інтерактивна аналітика прибутку та транзакцій з використанням графіків **Chart.js**.
* **🧾 Invoice Generation:** Автоматичне створення PDF-чеків з QR-кодами для підтвердження оплати.

## 🛠 Технологічний стек

* **Backend:** Python 3.10+, Django 4.2+
* **AI/OCR:** Tesseract OCR, Pytesseract, Pillow
* **Frontend:** HTML5, CSS3 (SCSS), Bootstrap 5, JavaScript
* **База даних:** SQLite (локально) / PostgreSQL (для деплою)
* **Звіти:** ReportLab (PDF generation)

## 🚀 Швидкий старт

1.  **Клонуйте репозиторій:**
    ```bash
    git clone [https://github.com/Pavlitoo/CarRental.git](https://github.com/Pavlitoo/CarRental.git)
    cd CarRental
    ```

2.  **Встановіть залежності:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Налаштуйте OCR (Tesseract):**
    Встановіть Tesseract OCR на вашу ОС та вкажіть шлях до `.exe` файлу у `views.py`.

4.  **Запустіть міграції та сервер:**
    ```bash
    python manage.py migrate
    python manage.py runserver
    ```
