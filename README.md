**CarRental** — це комплексна інтелектуальна веб-платформа для автоматизації процесів оренди автомобілів. Проєкт розроблений з акцентом на безпеку, автоматизовану перевірку клієнтів за допомогою технологій комп'ютерного зору (Computer Vision) та гнучку фінансову систему утримання клієнтів. 

Система проєктувалась як рішення "під ключ", що мінімізує ручну роботу адміністратора та надає користувачам безшовний і сучасний досвід бронювання.

---

## 🧠 Архітектура та Бізнес-логіка системи

Замість стандартних CRUD-операцій, цей проєкт реалізує складну бізнес-логіку, розділену на кілька ключових модулів.

### 🛡️ Модуль безпеки та AI-Верифікації (KYC)
Для доступу до автопарку кожен користувач повинен пройти багаторівневу перевірку. 
По-перше, система використовує підтвердження особи через одноразові паролі (OTP), які генеруються криптографічно безпечним методом та надсилаються на email клієнта за допомогою SMTP-сервера. Додатково реалізовано захищений механізм скидання забутого пароля з токенізованими посиланнями.
По-друге, реалізовано модуль Know Your Customer (KYC). Користувач завантажує фотографію своїх документів (паспорта або водійського посвідчення), після чого бекенд передає зображення до нейромережевого рушія **Tesseract OCR**. Алгоритм витягує весь текст, знаходить дати за допомогою регулярних виразів, вираховує точний вік клієнта і автоматично приймає рішення про допуск до сервісу (строго 18+).

### 💰 Фінансове ядро та Програма лояльності
Проєкт має вбудований віртуальний гаманець для кожного клієнта. Після кожної успішної поїздки система автоматично нараховує кешбек. 
Відсоток кешбеку динамічно змінюється залежно від VIP-статусу клієнта (Bronze — 5%, Silver — 7%, Gold — 10%), який розраховується на основі загальної суми витрат за весь час. Бали з гаманця можна використовувати для часткової або повної оплати наступних бронювань. Також реалізована система промокодів зі знижками та лімітами на кількість використань.

### 📑 Автоматизація документообігу
Для кожної транзакції бекенд генерує офіційний електронний чек у форматі PDF (за допомогою бібліотеки ReportLab). Чек містить повну деталізацію вартості, включаючи націнки за вихідні дні, застосовані промокоди та списаний кешбек. На кожен документ автоматично накладається унікальний QR-код та графічна печатка для підтвердження валідності.

### 🛠️ Управління автопарком (Fleet Management)
Система самостійно слідкує за технічним станом автомобілів. Бекенд відстежує кількість виконаних поїздок для кожної машини. Як тільки автомобіль завершує 5 поїздок, система автоматично змінює його статус на "Недоступний" і знімає з публікації, сповіщаючи адміністратора про необхідність проведення технічного обслуговування (ТО).

## ⚙️ Технологічний стек

**Бекенд:**
* Python (Core)
* Django (Web Framework & ORM)

**Фронтенд:**
* HTML5 / CSS3

* Vanilla JavaScript (DOM manipulation)

**Інфраструктура:**

* SMTP Backend (Email delivery)

## 📊 Панель адміністратора та Аналітика

Для власників бізнесу розроблено кастомний Dashboard. Він збирає дані з бази даних у реальному часі та візуалізує їх. Панель включає:
1. Загальний прибуток та кількість угод.
2. Суму наданих знижок (втрачена вигода).
3. Графік динаміки прибутку по днях за останні 2 тижні (інтерактивний лінійний графік).
4. Рейтинг Топ-5 найпопулярніших автомобілів автопарку.
5. Швидке управління станом автомобілів (скидання лічильників ТО в один клік).
